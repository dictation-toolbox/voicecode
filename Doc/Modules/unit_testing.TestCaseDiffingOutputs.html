Description

This class, captures STDOUT to a string and compares it to a benchmark file.

The test fails if the two files differ.

NOTE: All test methods of this class should: - invoke self.start_new_test() at the beginning - invoke self.compare_outputs() at the end

There must be a way to automate this, but I fought with pyUnit for a while and couldn't figure it out.
